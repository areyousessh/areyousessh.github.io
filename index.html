Projetos

Gameplay foi um projeto de um aplicativo movel (IOS / Android) desenvolvido em React-Native em uma NLW da Rocketseat. O Projeto foi feito utilizando Typescript como linguagem principal e diversos modulos nativos como Async Storage para a utilizacão do conceito de offline first, também foi utilizado conceitos de estilização especificos para disposivos mais modernos que contém notch na tela e conceitos comuns do React-Native como componentização, rotas e navegação e etc.
Dev Finances foi um projeto realizado em uma das primeiras NLW da Rocketseat. A aplicação basicamente faz um controle financeiro manual aonde você deve adicionar suas entradas e despesas e a aplicação vai calcular o seu saldo restante.
Nessa aplicação eu aprendi a utilizar a DOM e a utilizar calculos matemáticos utilizando Javascript
Esse aplicativo é um projeto pessoal aonde resolvi construir um aplicativo de agendamento de horários para meus amigos que tem uma barbearia. Ainda é um projeto em desenvolvimento que eu criei com o propósito de ajudar meus amigos mas também de aplicar tudo que vou aprendendo durante meus estudos e workshops que participo online.
